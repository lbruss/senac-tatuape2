# Ideia principal

Conceitos básicos do **Photoshop 2026**, principalmente sobre **camadas**, **pixels**, **vetores**, **modos de cor**, **seleção**, **pintura**, **apagamento**, **degradê**, **texturas** e **salvamento do projeto**.

A ideia central é entender que o Photoshop trabalha principalmente com **imagens formadas por pixels**. Para editar melhor essas imagens, eu uso **camadas**, que funcionam como folhas transparentes empilhadas. Cada camada pode ter uma parte diferente do projeto, permitindo editar uma coisa sem estragar o restante.

---

# O que são camadas no Photoshop?

As **camadas** são uma das partes mais importantes do Photoshop.

Uma camada funciona como uma folha transparente colocada sobre outra.
Cada elemento do projeto pode ficar em uma camada separada, como:

* uma imagem;
* uma cor;
* um texto;
* um desenho;
* um efeito;
* uma textura;
* um fundo.

Isso facilita muito a edição, porque eu consigo alterar uma parte do projeto sem mexer nas outras.

> Analogia

Posso imaginar as camadas como várias folhas de acetato uma em cima da outra.

Por exemplo:

* uma folha tem o fundo;
* outra folha tem o personagem;
* outra tem uma cor;
* outra tem um texto;
* outra tem um efeito.

Quando olho todas juntas, vejo a imagem completa.
Mas, se eu quiser mudar só o texto, eu mexo apenas na camada do texto.

## Atalhos básicos de zoom

Alguns atalhos importantes para trabalhar melhor com a imagem são:

`Ctrl + 0`

Esse atalho ajusta a imagem na tela, deixando ela em um zoom adequado para visualizar o projeto inteiro.

---

`Ctrl + +`

Esse atalho aumenta o zoom da imagem.

---

`Ctrl + -`

Esse atalho diminui o zoom da imagem.

---

**O Photoshop enxerga a imagem como pixels**

O Photoshop trabalha principalmente com imagens formadas por **pixels**.

Um **pixel** é o menor ponto de uma imagem digital.
Cada imagem é formada por milhares ou milhões de pequenos quadradinhos coloridos.

Quando se da muito zoom em uma imagem, começo a enxergar esses quadradinhos.
Isso acontece porque a imagem é formada por pixels.

- **Exemplo**

Uma foto tirada pelo celular é uma imagem de pixels.
Se eu aumentar muito essa foto, em algum momento ela começa a perder qualidade e ficar “quadriculada”.

Isso acontece porque os pixels ficam mais visíveis.

## Imagem rasterizada x vetor

No Photoshop, é importante entender a diferença entre **imagem rasterizada** e **vetor**.

---

**O que é uma imagem rasterizada??*

Uma imagem rasterizada é uma imagem formada por pixels.

Exemplos de imagens rasterizadas:

* fotos;
* prints;
* imagens baixadas da internet;
* arquivos JPG;
* arquivos PNG;
* imagens escaneadas.

Esse tipo de imagem tem limite de qualidade.
Se eu aumentar demais, ela pode ficar borrada ou pixelada.

- **Exemplo**

Se eu pego uma foto pequena e tento transformar em um banner muito grande, provavelmente ela perde qualidade.

Isso acontece porque o Photoshop precisa “esticar” os pixels que já existem.

---

##O que é vetor?##

O **vetor** é diferente da imagem comum.

O vetor não é feito diretamente de pixels.
Ele é feito com cálculos, linhas, curvas e formas matemáticas.

Por isso, um vetor pode ser aumentado ou diminuído sem perder qualidade.

- **Exemplo**

Um desenho vetorial pode ser usado pequeno em um cartão ou gigante em um outdoor, mantendo a qualidade.

Isso acontece porque o computador recalcula o desenho de acordo com o tamanho.

> Analogia

Uma imagem rasterizada é como uma parede feita de bloquinhos.
Se eu aumentar muito, começo a ver os blocos.

Já o vetor é como uma receita de desenho.
O programa não guarda apenas os bloquinhos, ele guarda as instruções para redesenhar aquilo em qualquer tamanho.

## RGB e CMYK

No Photoshop, também é importante entender os modos de cor.

Os principais são:

* **RGB**;
* **CMYK**.

---

**RGB — usado para telas**

O modo **RGB** é usado para imagens que serão exibidas em telas.

RGB significa:

```
R = Red / Vermelho
G = Green / Verde
B = Blue / Azul
```

Esse modo usa luz para formar as cores.

Exemplos de uso do RGB:

* sites;
* redes sociais;
* vídeos;
* telas de computador;
* celular;
* televisão;
* banners digitais;
* posts para Instagram;
* thumbnails.

**Por que RGB é usado em telas?**

Porque telas emitem luz.
A combinação de vermelho, verde e azul cria as outras cores que aparecem na tela.

---

**CMYK — usado para impressão**

O modo **CMYK** é usado para materiais impressos.

CMYK significa:

```
C = Cyan / Ciano
M = Magenta
Y = Yellow / Amarelo
K = Black / Preto
```

Esse modo trabalha com tinta.

Exemplos de uso do CMYK:

* panfletos;
* cartões de visita;
* banners impressos;
* cartazes;
* revistas;
* embalagens;
* documentos gráficos para impressão.

**Por que CMYK é usado para impressão?**

Porque impressoras usam tinta.
Enquanto a tela forma cores com luz, a impressão forma cores misturando pigmentos.

## Polegadas e centímetros

Uma medida importante usada em design é a polegada.

```
1 polegada = 2,54 cm
```

Essa medida aparece bastante quando se fala de:

* tamanho de papel;
* resolução;
* impressão;
* dimensões de imagem;
* DPI;
* configuração de documentos.

- **Exemplo**

Se um documento tem uma largura de 10 polegadas, isso significa:

```
10 x 2,54 = 25,4 cm
```

Ou seja, 10 polegadas equivalem a 25,4 centímetros.

# Ferramentas básicas usadas

**Varinha mágica**

A **Varinha Mágica** é uma ferramenta de seleção.

Ela serve para selecionar áreas com cores parecidas.

Normalmente, ela funciona melhor em regiões com cores sólidas ou com pouca variação de cor.

- **Exemplo**

Se eu tenho um desenho com fundo branco, posso usar a Varinha Mágica para selecionar esse fundo branco de forma rápida.

Depois de selecionar, posso apagar, pintar ou modificar aquela área.

---

**Como apagar uma área com a Varinha Mágica**

Para apagar uma área com cor sólida, o processo é:

```
Selecionar a Varinha Mágica
Clicar na área desejada
Pressionar Delete
```

Quando faço isso, a área selecionada é apagada.

Se a camada permitir transparência, aparece um fundo com quadrados cinza e branco.

Esse fundo quadriculado não faz parte da imagem.
Ele apenas indica que aquela região está transparente.

---

**O que significa o fundo quadriculado?**

O fundo quadriculado cinza e branco representa **transparência**.

Isso significa que não existe cor naquela parte da imagem.

### Exemplo

Se eu apago o fundo branco de uma imagem e aparece o quadriculado, significa que o fundo foi removido.

Quando eu exportar essa imagem como PNG, essa área pode continuar transparente.

---

## 1️⃣6️⃣ Atalho para tirar seleção

Depois de selecionar uma parte da imagem, posso remover a seleção usando:

```text
Ctrl + D
```

Esse atalho desfaz a seleção ativa.

### Por que isso é importante?

Porque, enquanto uma seleção está ativa, o Photoshop só permite editar aquela área selecionada.

Se eu tentar pintar ou apagar outra parte da imagem, pode parecer que a ferramenta não está funcionando, mas na verdade ela está limitada pela seleção.

Por isso, quando terminar de editar uma seleção, é bom usar:

```text
Ctrl + D
```

# Trabalhando com pintura e camadas

## 1️⃣7️⃣ Como criar uma nova camada

Para criar uma nova camada, eu clico no ícone de **+** no canto inferior direito, dentro do painel de camadas.

A nova camada aparece acima da camada selecionada.

### Por que criar uma nova camada?

Porque pintar em uma camada separada deixa o projeto mais organizado e seguro.

Se eu pintar diretamente na imagem original, posso estragar a imagem.
Mas, se eu pintar em uma nova camada, consigo apagar, mover ou modificar essa pintura sem afetar o desenho original.

---

## 1️⃣8️⃣ Pintando uma parte selecionada

Um processo básico para colorir uma imagem é:

```text
Selecionar uma área com a Varinha Mágica
Criar uma nova camada
Escolher uma cor
Selecionar o Balde de Tinta
Clicar na área selecionada
```

Esse processo permite colorir partes específicas da imagem.

### O que está acontecendo?

Primeiro, eu uso a Varinha Mágica para dizer ao Photoshop qual área quero editar.

Depois, crio uma nova camada para não pintar diretamente na imagem original.

Em seguida, escolho uma cor e uso o Balde de Tinta para preencher aquela área.

---

## 1️⃣9️⃣ Fluxo correto para colorir um desenho

O fluxo mais organizado é:

```text
Selecionar a parte desejada
Criar uma nova camada
Escolher a cor
Pintar com o Balde de Tinta
Repetir o processo para outras partes
```

Ou seja:

```text
Selecionar → Criar camada → Escolher cor → Pintar
```

Esse método ajuda a manter cada cor separada.

### Exemplo

Se eu estiver colorindo um personagem, posso separar assim:

* uma camada para a pele;
* uma camada para o cabelo;
* uma camada para a roupa;
* uma camada para o fundo;
* uma camada para sombras;
* uma camada para detalhes.

Assim, se eu quiser mudar só a cor da roupa, não preciso mexer no resto.

# Ferramenta Borracha

## 2️⃣0️⃣ Como apagar algo com a Borracha

Para apagar manualmente uma parte da imagem, uso a ferramenta **Borracha**, localizada na barra de ferramentas do lado esquerdo.

A Borracha remove pixels da camada selecionada.

### Atenção

A Borracha só apaga a camada que estiver selecionada.

Se eu tentar apagar e nada acontecer, provavelmente estou na camada errada.

---

## 2️⃣1️⃣ Como aumentar ou diminuir o tamanho da Borracha

Para mudar o tamanho da Borracha, uso os colchetes do teclado:

```text
[  diminui o tamanho da borracha
]  aumenta o tamanho da borracha
```

Esse atalho também funciona com outras ferramentas de pincel.

### Por que isso ajuda?

Porque eu consigo apagar áreas grandes com uma borracha maior e detalhes pequenos com uma borracha menor.

# Degradê

## 2️⃣2️⃣ O que é degradê?

O **degradê** é uma transição suave entre duas ou mais cores.

Em vez de preencher uma área com uma cor sólida, o degradê cria uma passagem gradual de uma cor para outra.

### Exemplo

Um fundo que começa azul e termina roxo é um degradê.

---

## 2️⃣3️⃣ Como usar o Degradê

Para usar o degradê:

```text
Ir até a ferramenta Balde de Tinta
Clicar e segurar sobre ela
Selecionar a ferramenta Degradê
Escolher as cores
Aplicar o degradê na imagem
```

No Photoshop, a ferramenta Degradê geralmente fica agrupada junto com o Balde de Tinta.

Ao clicar e segurar, aparecem outras ferramentas disponíveis no mesmo grupo.

---

## 2️⃣4️⃣ Escolhendo as cores do degradê

Na parte de cores, posso escolher uma cor para o início e outra para o final do degradê.

Por exemplo:

```text
Cor inicial: azul
Cor final: roxo
```

Depois, aplico o degradê arrastando na imagem.

A direção do arrasto influencia o resultado.

### Exemplo

Se eu arrastar de cima para baixo, o degradê será aplicado verticalmente.

Se eu arrastar da esquerda para a direita, o degradê será aplicado horizontalmente.

# Texturas e filtros

## 2️⃣5️⃣ Como aplicar textura usando filtros

Para aplicar uma textura, posso usar a Galeria de Filtros.

O caminho é:

```text
Selecionar a camada
Ir em Filtro
Clicar em Galeria de Filtros
Escolher o filtro desejado
Aplicar
```

Os filtros modificam visualmente a camada selecionada.

Eles podem criar efeitos como:

* textura de papel;
* granulado;
* pintura;
* distorção;
* traços artísticos;
* relevo;
* estilização.

---

## 2️⃣6️⃣ Por que selecionar a camada antes do filtro?

Porque o Photoshop aplica o filtro apenas na camada selecionada.

Se eu selecionar a camada errada, o efeito será aplicado no lugar errado.

Por isso, antes de usar qualquer filtro, é importante olhar no painel de camadas e confirmar qual camada está ativa.

# Inserindo outra imagem no projeto

## 2️⃣7️⃣ Como colocar outra imagem dentro do Photoshop

Para adicionar outra imagem no projeto, posso arrastar a imagem diretamente para dentro do Photoshop.

O processo é:

```text
Pegar a imagem no computador
Arrastar para dentro do Photoshop
Ajustar o tamanho e a posição
Confirmar a inserção
```

Depois disso, a imagem aparece como uma nova camada.

---

## 2️⃣8️⃣ Por que às vezes não consigo editar a imagem colocada?

Quando arrasto uma imagem para dentro do Photoshop, ela pode entrar como um **Objeto Inteligente**.

Um Objeto Inteligente protege a imagem original, permitindo redimensionar e transformar sem perder qualidade rapidamente.

Porém, algumas edições diretas nos pixels não funcionam enquanto a camada estiver nesse formato.

Por isso, se eu quiser editar diretamente essa imagem, preciso rasterizar a camada.

---

## 2️⃣9️⃣ O que é rasterizar uma camada?

Rasterizar significa transformar aquela camada em uma imagem comum de pixels.

Depois de rasterizar, eu consigo editar diretamente com ferramentas como:

* Borracha;
* Pincel;
* Balde de Tinta;
* Varinha Mágica;
* Filtros;
* ajustes diretos nos pixels.

O processo é:

```text
Selecionar a camada da imagem
Clicar com o botão direito
Escolher Rasterizar Camada
```

Depois disso, a imagem pode ser editada como uma camada comum.

### Atenção

Depois que rasterizo, perco parte da flexibilidade do Objeto Inteligente.

Por isso, só devo rasterizar quando realmente precisar editar os pixels diretamente.

# Salvando o projeto

## 3️⃣0️⃣ Como salvar o arquivo no formato Photoshop

Para salvar o projeto, sigo o caminho:

```text
Arquivo
Salvar como
Escolher o formato Photoshop
Salvar
```

O formato do Photoshop é:

```text
.psd
```

Esse formato mantém as camadas do projeto.

---

## 3️⃣1️⃣ Por que salvar em PSD?

O arquivo PSD é importante porque preserva a estrutura do projeto.

Ele mantém:

* camadas;
* textos editáveis;
* efeitos;
* transparências;
* ajustes;
* organização do arquivo.

Se eu salvar apenas como JPG, por exemplo, posso perder as camadas e o projeto vira uma imagem única.

### Exemplo

Se eu fiz um projeto com fundo, texto e imagem em camadas separadas, salvando em PSD eu consigo abrir depois e continuar editando cada parte.

Se eu salvar só em JPG, tudo vira uma imagem achatada.

# 🧩 Passo a passo geral da aula

## Criando e editando uma imagem no Photoshop

### 1. Abrir o Photoshop e adicionar uma imagem

Primeiro, adiciono uma imagem ao Photoshop para começar a edição.

Essa imagem será a base do projeto.

---

### 2. Organizar os painéis

Depois, vou na aba:

```text
Janela
```

E deixo visível o painel de:

```text
Camadas
```

Isso ajuda a acompanhar tudo que está sendo criado no projeto.

---

### 3. Usar o zoom para ajustar a visualização

Uso:

```text
Ctrl + 0
```

Para ajustar a imagem na tela.

Uso:

```text
Ctrl + +
```

Para aproximar.

Uso:

```text
Ctrl + -
```

Para afastar.

Isso facilita tanto a visualização geral quanto a edição de detalhes.

---

### 4. Selecionar áreas com a Varinha Mágica

Uso a Varinha Mágica para selecionar áreas de cor parecida.

Ela é útil principalmente para fundos brancos, áreas sólidas e desenhos com contornos definidos.

---

### 5. Apagar áreas selecionadas

Depois de selecionar uma área, posso pressionar:

```text
Delete
```

Isso apaga a área selecionada.

Se aparecer o fundo quadriculado, significa que aquela área ficou transparente.

---

### 6. Remover a seleção

Depois de terminar a edição, uso:

```text
Ctrl + D
```

Isso tira a seleção e libera a imagem para novas edições.

---

### 7. Criar uma nova camada

Clico no botão de:

```text
+
```

No painel de camadas.

Isso cria uma nova camada para trabalhar de forma separada.

---

### 8. Pintar com o Balde de Tinta

Depois de selecionar uma área e criar uma camada, escolho uma cor e uso o Balde de Tinta para preencher.

Esse processo ajuda a colorir sem alterar a camada original.

---

### 9. Usar Borracha quando necessário

Seleciono a Borracha para apagar manualmente partes da camada.

Uso:

```text
[  para diminuir
]  para aumentar
```

Assim consigo trabalhar com áreas grandes ou pequenos detalhes.

---

### 10. Aplicar degradê ou textura

Se quiser um efeito mais elaborado, posso usar:

```text
Degradê
```

Ou ir em:

```text
Filtro → Galeria de Filtros
```

Isso deixa o projeto com mais estilo visual.

---

### 11. Adicionar outra imagem

Posso arrastar outra imagem para dentro do Photoshop.

Se precisar editar diretamente essa imagem, posso rasterizar a camada.

---

### 12. Salvar o projeto

Por fim, salvo como:

```text
Arquivo Photoshop (.psd)
```

Assim mantenho as camadas e consigo editar depois.
