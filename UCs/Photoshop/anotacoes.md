# Máscaras Visuais e Ferramenta Caneta

**Ideia principal**

Criar uma composição visual mais profissional utilizando várias imagens juntas. Trabalhei com **remoção de fundo**, **seleção automática de objetos**, **mistura de imagens usando borracha suave**, **organização de camadas**, **efeitos de profundidade** e também comecei a utilizar uma das ferramentas mais importantes do Photoshop: a **Ferramenta Caneta (Pen Tool)**.

O objetivo foi entender como combinar várias imagens em uma única arte, criando um resultado visual mais elaborado do que simplesmente colocar imagens lado a lado.

---

# Cartão de Cafeteria

**Objetivo do projeto**

Criar um cartão promocional de cafeteria utilizando:

* uma imagem de grãos de café;
* uma imagem de uma xícara de café;
* uma imagem de pôr do sol;
* textos e elementos decorativos.

O resultado final é uma composição com profundidade e aparência profissional.

---

**Organizando as imagens**

Foram utilizadas três imagens:

- **Imagem 1**

```
Grãos de café
```

Será usada como textura principal.

---

- **Imagem 2**

```
Xícara de café
```

Será o objeto principal da composição.

---

- **Imagem 3**

```
Pôr do sol
```

Será utilizada como plano de fundo.

---

**Como funciona a organização das camadas?**

No Photoshop, as camadas funcionam como folhas empilhadas.

- **Exemplo**

```
Xícara de café
↓
Grãos de café
↓
Pôr do sol
```

Aquilo que está mais acima aparece na frente.

Aquilo que está mais abaixo aparece atrás.

---

**Ferramenta Seleção de Objeto**

A Ferramenta Seleção de Objeto é uma evolução das ferramentas tradicionais de seleção.

Ela utiliza inteligência artificial para identificar automaticamente objetos dentro da imagem.

Ela fica agrupada com outras ferramentas de seleção.

- **Caminho**

```
Varinha Mágica
↓
Segurar clique
↓
Ferramenta Seleção de Objeto
```

---

**O que ela faz?**

Ela tenta reconhecer automaticamente:

* pessoas;
* animais;
* carros;
* copos;
* xícaras;
* objetos em geral.

Isso economiza muito tempo comparado a fazer uma seleção manual.

---

**Selecionando a xícara**

Após selecionar a Ferramenta Seleção de Objeto:

```
Selecionar a xícara
```

O Photoshop identifica automaticamente os limites do objeto.

---

**O que acontece?**

A seleção inicial normalmente escolhe:

```
Xícara
```

Mas o objetivo da atividade é remover o fundo.

Por isso foi necessário inverter a seleção.

---

**Invertendo a seleção**

Atalho utilizado:

```
Ctrl + Shift + I
```

---

**O que esse atalho faz?**

Ele inverte completamente a seleção.

- **Antes**

```
Selecionado:
Xícara
```

- **Depois**

```
Selecionado:
Fundo
```

---

**Por que inverter?**

Porque queremos apagar apenas o fundo.

Ao inverter, o Photoshop passa a selecionar tudo que NÃO é a xícara.

---

**Removendo o fundo**

Depois de inverter a seleção:

```
Delete
```

---

**Resultado**

O fundo desaparece.

Permanece apenas:

```
Xícara de café
```

com transparência ao redor.

---

**Como saber que deu certo?**

Aparecerá o padrão quadriculado do Photoshop.

Esse padrão representa:

```
Transparência
```

---

**Trabalhando a imagem dos grãos de café**

Agora começamos a criar um efeito visual mais interessante.

Selecionamos a camada:

```
Grãos de café
```

---

**Ferramenta utilizada**

```
Borracha
```

---

**Configuração**

```
Dureza: 0%
```

---

**O que significa dureza 0%?**

Uma borracha com dureza zero possui bordas extremamente suaves.

Ela não cria cortes bruscos.

> Analogia

Imagine apagar um desenho:

* com uma borracha comum → borda dura;
* com um pincel esfumaçado → borda suave.

A dureza 0% se aproxima do segundo caso.

---

**Criando uma transição suave**

Após configurar a borracha:

```
Apagar gradualmente a parte superior
```

---

**Objetivo**

Não é apagar totalmente.

É criar uma transição visual.

Isso faz com que os grãos pareçam desaparecer aos poucos.

---

**Resultado visual**

Em vez de uma divisão brusca:

```
Grãos
==========
Nada
```

Temos:

```
Grãos
↓↓↓↓↓↓
Transição
↓↓↓↓↓↓
Fundo
```

---

**Trabalhando a opacidade da Borracha**

A ferramenta Borracha também possui controle de opacidade.

---

**O que é opacidade da borracha?**

Controla a intensidade do apagamento.

- **Exemplo**

```
100% = apaga completamente
```

```
50% = apaga parcialmente
```

---

**Aplicação na atividade**

Foram dados alguns cliques suaves na parte inferior dos grãos.

Isso ajuda a misturar melhor os elementos.

---

**Adicionando o pôr do sol**

Agora foi adicionada uma terceira imagem.

```
Pôr do sol
```

---

**Posição correta**

A camada deve ficar:

```
Última camada
```

Ou seja:

```
Embaixo de todas as outras
```

---

**Estrutura final**

```
Xícara
↓
Grãos de café
↓
Pôr do sol
```

---

**Por que colocar o pôr do sol atrás?**

Porque ele funciona como plano de fundo.

Se ele ficar acima:

* esconderá os grãos;
* esconderá a xícara.

**Conceito Fundamental**

## Composição visual?

Composição visual é a forma como os elementos são organizados dentro de uma arte.

Ela define:

* equilíbrio;
* hierarquia;
* destaque;
* harmonia.

Uma boa composição faz o olhar da pessoa percorrer naturalmente a arte.

# Ferramenta Caneta

**Introdução**

A Ferramenta Caneta é uma das ferramentas mais importantes do Photoshop.

Muitos iniciantes têm dificuldade com ela porque funciona de maneira diferente das outras ferramentas.

Porém, ela oferece uma precisão muito maior.

---

**O que é a Ferramenta Caneta?**

A Ferramenta Caneta serve para criar:

* linhas;
* curvas;
* contornos;
* caminhos vetoriais;
* seleções extremamente precisas.

Ela é muito utilizada para:

* recortes profissionais;
* logos;
* ilustrações;
* design gráfico;
* tratamento de imagens.

---

**Verificando o modo correto**

Antes de começar, verificar na barra superior:

```text
Caminho
```

---

**Por que usar Caminho?**

Porque nesse modo a Caneta cria trajetos vetoriais.

Esses trajetos podem ser transformados posteriormente em seleções precisas.

---

**Criando linhas retas**

Para criar uma linha reta:

```
Clique em um ponto
Clique em outro ponto
```

---

**O que acontece?**

O Photoshop cria automaticamente uma linha reta entre os dois pontos.

Esses pontos são chamados de:

```
Pontos de ancoragem
```

---

**Criando curvas**

Para criar curvas:

```
Clique
Segure o botão do mouse
Arraste
```

---

**O que acontece?**

A Caneta cria alças de controle.

Essas alças determinam:

* direção da curva;
* intensidade da curva;
* suavidade da curva.

---

> Analogia

Imagine que a linha é um arame flexível.

Os pontos de ancoragem são os lugares onde você segura o arame.

As alças são as forças que puxam esse arame para criar a curvatura.

---

**Por que a Caneta é tão importante?**

Porque ela oferece mais precisão que:

* Laço;
* Varinha Mágica;
* Seleção de Objeto.

Em imagens difíceis, a Caneta normalmente produz o melhor recorte.

---

- **Exemplos de uso profissional**

* Recorte de pessoas;
* Produtos para e-commerce;
* Logotipos;
* Embalagens;
* Vetorização;
* Design publicitário.
