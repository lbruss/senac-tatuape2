# 💻 Curso Técnico em Informática - SENAC Tatuapé

Repositório dedicado ao registro de **conteúdos, práticas, anotações e projetos** desenvolvidos ao longo do Curso Técnico em Informática do SENAC Tatuapé.  
Os materiais estão organizados por temas, com atualizações contínuas conforme o avanço dos estudos.

---

## 📚 Conteúdos Estudados
- Teoria de Hardware  
- Elétrica básica  
- Eletrônica básica com **Arduino**  
- Versionamento de código com **Git e GitHub**  
- Introdução à **Inteligência Artificial (IA)**  
- Sistemas Operacionais **Microsoft Windows**  
- Sistemas Operacionais **GNU/Linux** (Open Source)  
- Procedimentos de instalação, configuração e documentação técnica
- Redes
- Linux Server
- Windows Server
---

## 🎯 Objetivo do Repositório
Este espaço funciona como:
- **Central de estudos:** resumos, anotações e materiais de apoio.  
- **Registro de práticas:** exercícios, comandos e projetos desenvolvidos nas aulas.  
- **Guia rápido:** documentação de procedimentos que podem auxiliar outros estudantes.

---

## 📌 Observação
Repositório em constante atualização conforme novos conteúdos são estudados e documentados.
